- Int
- Float
- Boolean
Python e javascript seguem esta 'filosofia'.

> Obs: O tipo long, ao atribuir um valor numérico, é necessário colocar um l(Maiúsculo ou Minúsculo) no final do número.
> Ex:
> `long MyNumber = 10032399372L;`


Outra observação
> Podemos deixar os números grandes mais legíveis colocando um underline ( _ ) como separação
> Ex:
> `long MyOtherNumber = 123_212_333L;

O Casting em Java também é um pouco diferente, a sua sintaxe é:
